# Ninja Animation - Unity

## 📖 Description

This project was developed as part of a university animation assignment using **Unity**.

The objective was to create a cinematic combat animation by combining motion capture, 3D modeling, and visual effects.

## ✨ Features

- 🥷 Animated ninja character.
- 🗡️ Custom katana modeled by me.
- 🎥 Motion capture animation using body tracking.
- ✨ Particle effects created directly in Unity.
- 🎮 Real-time scene rendered in Unity.

## 🛠️ Technologies

- Unity
- Motion Capture (Body Tracking)
- Blender (Katana Modeling)
- Unity Particle System

## 📂 Project Structure

```
Assets/
Packages/
ProjectSettings/
```

## ▶️ How to Open the Scene

After opening the project in Unity, the main scene can be found at:

```
Assets/Scenes/SampleScene.unity
```

Open **SampleScene.unity** to view and run the complete animation.

## 🎬 Project Overview

The animation was created using a downloaded ninja character model combined with a katana designed and modeled by me. The character's movements were recorded using a motion capture suit, allowing me to perform the animation myself. Visual effects were created with Unity's built-in Particle System to enhance the final scene.
